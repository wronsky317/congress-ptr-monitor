# Pending Long-Term Updates

This directory stores auto-generated, reviewable candidate updates for
`long_term_views/congress_ptr_signal_map.md`.

Daily runs create one file per Asia/Shanghai date:

```text
long_term_views/pending_updates/YYYY-MM-DD.md
```

These files are commit-safe summaries. They should not contain raw PDF text,
full daily reports, Feishu chat ids, credentials, or private runtime state.
Review and merge the durable parts into the main signal map manually.
